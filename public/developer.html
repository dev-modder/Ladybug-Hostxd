How to Add Your Bot Script

Your bot script will be fetched from GitHub and saved on the server. Users can then assign it to their sessions. The script is executed with Node.js and receives session data via environment variables: SESSION_ID, BOT_NAME, OWNER_NUMBER, PREFIX, TIMEZONE.

1
Create a public GitHub repository with your bot script. The entry point must be index.js.
2
Navigate to your file on GitHub and copy the URL — e.g. https://github.com/you/repo/blob/main/index.js
3
Click "🔗 Add via GitHub URL" below, paste the URL, and submit.
4
Users can assign your bot to their sessions from the Dashboard. You can push updates anytime — just click ↻ Pull Update.
